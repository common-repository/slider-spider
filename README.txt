== Description ==

Simple slider plugin.Its a beta version plugin. More features await to action.

== Installation ==

1. Upload `slider-spider.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

How to add gallery? 

Simply go to add new slider > add media > create gallery > insert gallery(full size)

 How to find shortcodes? 
click slider-spider menu and get your desired shortcode
